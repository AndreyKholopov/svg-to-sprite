Работа с SVG спрайтами

Создание спрайта

Символьный svg-спрайт, представляет собой один корневой элемент <svg>, внутри которого создаются области <symbol>, внутри которых содержится код svg-иконки.Для создания SVG спрайтов нужно просто объеденить разные SVG изображения в один тег <svg> и расставить id в тегах <symbol>. Также в спрайтах перестают работать внутренние теги <defs>, поэтому при создании спрайтов лучше убрать теги <defs> и убрать все ссылки, типа fill: url(#example).

Для создания спрайтов можно использовать автоматизированные системы, например svgsprit.es.

Вызов определенного SVG из спрайта

Вызов из спрайта определленной svg картинки можно осуществить в теге <use> после ссылки на файл спрайта указать id нужного svg изображения. Пример: xlink:href="exampleName.svg#example-id".

Управление свойствами SVG с помощью currentColor

Если в редактировании SVG нужно изменять только 2 цвета, то можно обойтись использованием currentColor. Для этого нужно удалить атрибуты fill для задания основного цвета и добавить атрибут fill, со значением currentColor для задания дополнительного цвета. После этого, в CSS указать два значения: fill для цвета основной иконки и color для цвета ее второй части. Данный способ поддерживается всеми современными браузерами.

Управление свойствами SVG с помощью CSS-переменных

Для указания своего свойства из CSS можно использовать переменные var. В svg нужно использовать переменные, например: fill="var(--example-color)" и изменять уже переменную в css. Данный способ не будет работать в некоторых версиях IE, Opera Mini и Android Browser, так как в них отсутствует поддержка CSS-переменных.

Таким способом невозможно задать градиент, но его можно задать непосредственно внутри тега <svg>, что в файле html, также, как и в <defs>.
Пример:

<svg>
   <linearGradient id="Gradient01">
      <stop offset="20%" stop-color="#39F" />
      <stop offset="90%" stop-color="#F3F" />
   </linearGradient>
   <use ...>
