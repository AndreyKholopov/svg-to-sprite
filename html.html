Работа с SVG спрайтами

Создание спрайта

Для создания SVG спрайтов нужно просто объеденить разные SVG изображения в один тег <svg> и расставить id в тегах <symbol>. Также в спрайтах перестают работать внутренние теги <defs>, поэтому при создании спрайтов лучше убрать теги <defs> и убрать все ссылки, типа fill: url(#example).

Также для создания спрайтов можно использовать автоматизированные системы, например svgsprit.es.

Вызов определенного SVG из спрайта

Для вызова из спрайта определленной svg картинки, нужно в теге <use> после ссылки на файл спрайта указать id нужного svg изображения. Пример: xlink:href="exampleName.svg#example-id".

Управление свойствами svg с помощью css

Для указания своего свойства из CSS можно использовать переменные var. В svg нужно указать использовать переменные, например: fill="var(--example-color)" и изменять уже переменную в css. Таким способом невозможно задать градиент, но его можно задать непосредственно внутри тега <svg> также, как и в <defs>.
Пример:

<linearGradient id="Gradient01">
   <stop offset="20%" stop-color="#39F" />
   <stop offset="90%" stop-color="#F3F" />
</linearGradient>
